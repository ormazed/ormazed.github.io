## 01. 인터페이스 기반의 프로그래밍이란
---
(ex)
어플리케이션과 DB 를 연결하기 위해서는 DB 에서 제공하는 API 를 사용하게 된다.
이 연결을 위해 인터페이스를 사용하는데, 다수의 DB 벤더사의 각기 다른 API 에 영향을 받지 않고 사용할 수 있게 된다.

인터페이슬르 사용하는 이유 : class 내부의 동작방식에 영향을 받지 않고도 class 를 사용할 수 있도록 하기 위함
-> 느슨한 연결이 가능하도록 함.
