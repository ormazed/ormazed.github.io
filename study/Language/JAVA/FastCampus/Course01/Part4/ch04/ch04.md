# CH04. 다형성과 활용
## 01. 다형성이란 무엇인가
---
* (message) polymorphism : message 에 대하여 다양하게 반응하는 것
: 상위클래스(Animal)가 동일한 메시지(eat())로 하위클래스(Dog, Cat)를 서로 다르게 동작 시키는 객체지향이론
```
Animal ani;
ani = new Dog();
ani.eat();
ani = new Cat();
ani.eat();

// 강이지처럼 먹는다
// 고양이처럼 먹는다.  
```
=> PolyTest.java
