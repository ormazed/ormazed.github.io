# ssh 통신시 비밀전호 없이 통신하는 방법

'''
<src 서버에서 작업>
ssh-keygen
'''
- 엔터 3회
- 현재 로그인된 계정의 홈경로₩.ssh 에 공개키,개인키 가 생긴다

'''
<src 서버에서 작업>
ssh-copy-id 계정@원격지IP
'''
- dest 서버의 계정 홈경로₩.ssh 이하에 authori~라는 명의 파일이 생긴다
- 해당 파일 내용을 보면 src서버의 공개키 내용이 보인다

### 주의   
- 홈 디렉터리의 퍼미션이 700이상이면 위의 작업을 해도 계속 비밀번호를 묻는다
-- 홈디렉터리 700
-- .ssh 디렉터리 700
-- 공개키/암호키 644
